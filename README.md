# Rocket Boost 🚀
A simple 3D arcade game built with Unity.
To see the core logic I developed for this project, please refer to the following path: Assets/Scripts


## How to Play
* Navigate the rocket through obstacles.
* Reach the landing pad to advance to the next level.
* Complete all 3 levels to win!
* Press 'C' to toggle collision detection for testing and 'L' to instantly advance to the next level.
